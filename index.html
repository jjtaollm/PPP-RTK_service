Technical notes

Data access and product specification

The table summarizes the access entries and product information for the real-time satellite-based correction service. Using a regional tracking network in China, the service provides real-time multi-GNSS, multi-frequency orbit, clock, code bias, and phase bias products for the China region, covering GPS L1/L2/L5, Galileo E1/E5a/E5b, and BDS B1I/B3I/B2a/B1C signals. Products are available in both center-of-mass (CoM) and antenna phase-center (APC) forms.
